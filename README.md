### Git alias to use git effectively

#### Installation
```
> npm install -g lazzygit
```

#### Usage  

| Git command  | Lazzygit command |
|--------------|------------------|
| git init     | lg i             |
| git status   | lg s             |
| git pull     | lg pl            |
| git push     | lg ps            |
| git branch   | lg b             |
| git commit   | lg cm            |
| git add      | lg a             |
| git log      | lg l             |
| git diff     | lg d             |
| git reset    | lg r             |
| git merge    | lg m             |
| git checkout | lg co            |
| git stash    | lg ss            |
| git clone    | lg cl            |
| git tag      | lg t             |
| git remote   | lg rmt           |
| git apply    | lg apl           |
